### Selected field notes

- **[forum](https://github.com/KYUPPIDO/forum)** &middot; go &middot; sqlite &middot; docker<br>
  A discussion platform built from scratch: bcrypt auth, UUID sessions, posts, comments, votes, and enough structure to keep the noise out.
- **[net-cat](https://github.com/KYUPPIDO/net-cat)** &middot; go &middot; tcp &middot; goroutines<br>
  The classic `nc` rebuilt as a group chat: one server, many clients, pure standard library, no shortcuts.
- **[lem-in](https://github.com/KYUPPIDO/lem-in)** &middot; go &middot; graphs &middot; pathfinding<br>
  A digital ant farm: disjoint shortest paths, colony flow, and the kind of problem that rewards patient reasoning.
- **groupie-tracker** &middot; go &middot; rest apis &middot; openstreetmap<br>
  Concert discovery with server-side merging and a geocoded venue map. The longer write-up lives on [the portfolio](https://kyuppido.github.io/portfolio/).
